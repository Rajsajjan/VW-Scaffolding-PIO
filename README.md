# VW Scaffolding PIO

A Python-based Vectorworks 2026 Plug-in Object (PIO) for generating Layher Allround scaffolding.

## Planned Features

- Parametric scaffold
- Bay Length / Bay Width
- Multiple Bays
- Multiple Levels
- Layher Base Collar
- Layher Adjustable Base
- Layher Standards
- Layher O-Ledgers
- Diagonals
- Steel Decks
- Guardrails
- Bill of Materials
